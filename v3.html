Decisions coding test for JNU MCA 2019

Instructions:

  1. All questions are mandatory.
  2. You will be given 2 hours to complete the test.
  3. The answers will be evaluated based on following parameters
  • a. Correctness
  • b. Completeness
  • c. Readability
  • d. Maintainability

Exercise:

Q1. Write a program that takes an array as input n integers as input and prints the difference between the sum of numbers at even indices and sum of numbers at odd indices.

Example:
Input: [1, 5, 11, 13, 8, 20]
Output: -18

Q2. Write a program that takes a string as input and prints all the unique alphabets in that string, along with their count in the increasing order.

Example:
Input: "Sequence"
Output:
S-1
q-1
u-1
n-1
c-1
e-3

Q3. Given N number of bottles in which one bottle is poisoned. So the task is to find out minimum number of rats required to identify the poisoned bottle. A rat can drink any number of bottles at a time.

Example:
Input: N = 4
Output: 2

Input: N = 100
Output: 7
